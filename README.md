# 我的世界 MCPatcher 资源包中文教程

### [开始阅读](book/menu.md)

在`MCPatcher`中提供了许多功能，通过它，资源包作者可以实现出各种效果。

新版本的`OptiFine`实现了`MCPatcher`中的大部分功能，使用`OptiFine`可以大致代替`MCPatcher`；在本篇教程中，除另外声明外，使用`OptiFine`与`MCPatcher`的效果是大致相同的。
