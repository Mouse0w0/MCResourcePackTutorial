# 中文Minecraft资源包制作教程

### [开始阅读](book/menu.md)

Minecraft资源包提供了一系列用于修改游戏内资源的途径。通过使用资源包，玩家可以修改游戏内的贴图，音效和文本等信息。通过本文，你可以了解到如何制作资源包，还可以学习到使用`MCPatcher`提供的高级功能进一步修改游戏资源。
